## Notice

TerraLuna Launcher is currently under active development.

A public release may be provided after development is completed.

The launcher itself is planned to be available free of charge. However, depending on future circumstances, supporters may receive additional benefits such as remote assistance, installation support, setup guidance, and usage assistance.

As the project is still in development, the source code is currently private and will not be disclosed.

Thank you for your interest and support.

---

## 안내

TerraLuna Launcher는 현재 개발이 진행 중인 프로젝트입니다.

개발 완료 후 일반 배포가 진행될 예정입니다.

런처 자체는 무료로 제공될 예정이나, 향후 프로젝트 운영 상황에 따라 후원자에게 원격 지원, 설치 지원, 설정 안내, 사용법 안내 등의 추가 지원이 제공될 수 있습니다.

현재는 개발 단계에 있으므로 소스 코드는 공개되지 않습니다.

프로젝트에 관심을 가져주셔서 감사합니다.
