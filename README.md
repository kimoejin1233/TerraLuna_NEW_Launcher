## Notice

TerraLuna Launcher is currently under active development.

A public release may be provided after development is completed. However, depending on future circumstances, distribution may be offered through a supporter-based model.

As the project is still in development, the source code is currently private and will not be disclosed.

Thank you for your interest and support.

## 안내

TerraLuna Launcher는 현재 개발이 진행 중인 프로젝트입니다.

개발 완료 후 일반 배포가 진행될 수 있으나, 향후 프로젝트 운영 상황에 따라 후원 기반 방식으로 제공될 수 있습니다.

현재는 개발 단계에 있으므로 소스 코드는 공개되지 않습니다.

프로젝트에 관심을 가져주셔서 감사합니다.
